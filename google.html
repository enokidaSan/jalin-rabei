Principais comandos para criar um site.

Comandos necessarios do HTML:

Html: Define o início e o fim de um documento HTML.

Head: Contém metadados e links para recursos externos.

Title: Define o título da página.

Body: Contém o conteúdo visível da página.

h1, h2, h3, etc.: Definem os níveis de cabeçalhos.

p: Define um parágrafo.

a: Cria um link.

Img: Insere uma imagem.

Div: Agrupa elementos para estilo e layout.

Principais comandos do Css:

selector { property: value; }: Define regras de estilo para elementos selecionados.

color: Define a cor do texto.

background-color: Define a cor de fundo.

font-size: Define o tamanho da fonte.

margin: Define a margem ao redor dos elementos.

padding: Define o espaço interno ao redor do conteúdo de um elemento.

border: Define a borda de um elemento.

display: Controla como o elemento é exibido (block, inline, flex, etc.).
